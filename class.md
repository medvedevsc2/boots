# Глоссарий

| Термин | Определение |
|:--|:--|
| User |	Класс, в котором описано общее поведение для клиентов и работников
| Customer |	Класс, наследующий User, представляющий клиента магазина. Хранит информацию о заказах клиента, истории покупок и предпочтениях. |
| Manager |	Класс, наследующий User, представляющий сотрудника магазина. Содержит информацию о должности, списке обработанных заказов и правах доступа. |
| Administrator |	Класс, наследующий User, представляющий администратора системы. Управляет пользователями и системными настройками. |
| Product |	Класс, описывающий модель кроссовок. Содержит характеристики: бренд, модель, цену, размеры и описание. |
| Sneakers| 	Класс, представляющий конкретную пару кроссовок на складе. Содержит информацию о размере, цвете, количестве и уникальном идентификаторе. |
| Order |	Класс, описывающий заказ клиента. Содержит данные о заказанных кроссовках, их количестве, статусе заказа и суммарной стоимости. |
| Cart |	Класс, описывающий корзину покупок клиента. Содержит выбранные товары, их количество и общую стоимость. |
| Inventory |	Класс, управляющий запасами товаров. Отслеживает количество доступных пар для каждой модели и размера. |
